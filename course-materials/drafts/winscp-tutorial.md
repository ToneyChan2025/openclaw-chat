# WinSCP 使用教程

## 简介
WinSCP 是 Windows 上最好的 SFTP/SCP 客户端，用于在本地电脑和云服务器之间传输文件。

## 安装
1. 访问 https://winscp.net
2. 下载安装包
3. 按提示安装

## 配置连接

### 连接信息
| 字段 | 值 |
|------|-----|
| 文件协议 | SFTP |
| 主机名 | 43.130.46.144 |
| 端口号 | 22 |
| 用户名 | root |
| 密码 | 你的服务器密码 |

### 步骤
1. 打开 WinSCP
2. 填写上述连接信息
3. 点击"保存"保存会话
4. 点击"登录"连接


## 基本操作

### 下载文件
- 从右侧（服务器）拖到左侧（本地）
- 或右键点击文件 → 下载

### 上传文件
- 从左侧（本地）拖到右侧（服务器）
- 或右键点击文件 → 上传

### 编辑文件
- 双击文件，会用本地编辑器打开
- 保存后自动上传回服务器

1. **保存会话**：方便下次快速连接
2. **同步浏览**：左右两侧同步切换目录
3. **命令行**：Ctrl+T 打开终端
