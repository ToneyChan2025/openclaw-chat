# 个人日记系统

## 概述

智能个人日记系统，支持自动梳理、天气记录、标题摘要生成。

## 触发方式

- `个人日记`
- `写日记`
- `日记`
- `记录：xxx`

## 固定日记格式

```markdown
# 日记 - 2026-02-14

## 📋 日记信息
- **日期**: 2026-02-14 星期六
- **天气**: 广州 多云 22°C 湿度65%
- **地点**: 广州

## 📝 标题
（根据内容生成，20字以内）

## 📄 摘要
（根据内容生成，100字以内）

## 🖊️ 日记内容

### 上午

### 下午

### 晚上

## 💭 感想与反思

---
*最后更新: 14:45*
```

## 日记原则

1. **智能梳理**: 当天首次启动时梳理昨日日记
2. **天气记录**: 默认广州天气（可修改地点）
3. **标题生成**: 根据内容自动生成（20字内）
4. **摘要生成**: 根据内容自动生成（100字内）
5. **主动记录**: 需用户主动发起

## 使用方法

### 开始记录
```
个人日记
```

### 添加记录
```
记录：今天完成了重要工作
记录上午：开了项目会议
记录下午：和家人聚餐
```

### 生成标题摘要
```
生成标题和摘要
```
